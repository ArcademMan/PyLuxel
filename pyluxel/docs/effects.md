# Effects

Lighting, fog, particles, and scene transitions.

## LightingSystem

2D lighting with point lights, spotlights, falloff modes, flicker effects, and normal mapping. Supports up to **256 lights** batched in a single draw call.

### Basic Usage

```python
from pyluxel import Light, LightingSystem, FalloffMode

lighting = LightingSystem(ctx, renderer.light_prog)

# Add lights
torch = lighting.add(
    x=400, y=300,
    radius=200,
    color=(1.0, 0.8, 0.5),
    intensity=1.2,
)

# Each frame (inside the rendering pipeline)
renderer.begin_lights()
lighting.clear()
lighting.add(player_x, player_y, radius=150)
lighting.render(time=app.time)  # time enables flicker
```

### Light Parameters

```python
light = Light(
    x=400, y=300,             # position (design space)
    radius=150.0,             # light radius
    color=(1.0, 1.0, 0.9),    # RGB color (0-1)
    intensity=1.0,            # brightness multiplier
    falloff=FalloffMode.QUADRATIC,  # LINEAR, QUADRATIC, or CUBIC
    z=70.0,                   # height above surface (for normal mapping)
)
```

### Spotlights

```python
spot = Light(
    x=400, y=100,
    radius=300,
    color=(1.0, 1.0, 1.0),
    intensity=1.5,
    is_spotlight=True,
    direction=90.0,   # degrees (0=right, 90=down)
    angle=45.0,       # cone width in degrees
)

# Or configure an existing light
light.set_spotlight(direction=90.0, angle=30.0)
light.set_direction(180.0)  # point left
```

### Falloff Modes

```python
from pyluxel import FalloffMode

# FalloffMode.LINEAR    = 0  (gentle, large soft area)
# FalloffMode.QUADRATIC = 1  (default, realistic)
# FalloffMode.CUBIC     = 2  (harsh, concentrated center)

light.set_falloff(FalloffMode.CUBIC)
```

### Flicker

```python
light.set_flicker(
    speed=8.0,        # oscillation speed (0=off, 8=torch, 30=neon)
    amount=0.3,       # intensity variation (0-1, 0.2=subtle, 0.5=obvious)
    style="candle",   # "smooth" (sine), "harsh" (broken neon), "candle" (organic)
)
```

### Normal Mapping

Render normal map textures in the normal FBO to enable per-pixel lighting.

```python
# 1. Scene pass
renderer.begin_scene()
batch.begin(diffuse_texture)
batch.draw(x, y, w, h)
batch.end()

# 2. Normal map pass (same positions as scene)
renderer.begin_normal()
batch.begin(normal_map_texture)
batch.draw(x, y, w, h)
batch.end()

# 3. Lights pass (normals are automatically used)
renderer.begin_lights()
lighting.render(time)
```

Normal map format: tangent-space (R=X, G=Y, B=Z). Default color (128, 128, 255) = flat surface facing the camera.

### Shadow Casting

Lights can cast dynamic 2D shadows from occluder geometry. Shadow-casting lights use a 1D polar shadow map atlas generated via raymarching. Up to **64 shadow lights** are supported and rendered in a single batched draw call; excess lights are rendered without shadows.

```python
# Enable shadows on a light
light = Light(
    x=400, y=300, radius=250,
    color=(1.0, 0.9, 0.7), intensity=1.5,
    cast_shadows=True,        # enable shadow casting
    shadow_softness=0.03,     # penumbra size (0.01=hard, 0.1=soft)
)

# Or toggle on an existing light
light.set_shadow_casting(enabled=True, softness=0.02)
```

Shadow-casting requires drawing occluders (walls, obstacles) in a separate pass:

```python
# Manual pipeline
renderer.begin_scene()
batch.begin(texture)
batch.draw(...)  # scene
batch.end()

renderer.begin_shadow_casters()  # NEW: draw occluders here
batch.begin(white_tex)
batch.draw(wall_x, wall_y, wall_w, wall_h)  # white = blocks light
batch.end()

renderer.begin_lights()
lighting.render(time)  # shadow maps generated automatically
```

With the App class:

```python
# Decorator pattern
@app.on_shadow_casters
def shadow_casters():
    app.draw_rect(wall_x, wall_y, wall_w, wall_h, r=1, g=1, b=1)

# Subclass pattern
class MyGame(App):
    def shadow_casters(self):
        for wall in self.walls:
            self.draw_rect(wall.x, wall.y, wall.w, wall.h, r=1, g=1, b=1)
```

#### Shadow Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| `cast_shadows` | `False` | Enable shadow casting (max 64 lights) |
| `shadow_softness` | `0.02` | Penumbra size (0.01 = hard edge, 0.1 = very soft) |

#### How It Works

1. Occluders are drawn to an **occlusion FBO** (white = blocks light)
2. For each shadow light, a **1D shadow map** (360x1) is generated by raymarching from the light center at each angle
3. The light shader samples the shadow map to determine if a fragment is occluded
4. Non-shadow lights continue to render in the existing batch (zero overhead)

### Light Methods

| Method | Description |
|--------|-------------|
| `set_position(x, y)` | Move light |
| `set_color(r, g, b)` | Change color (RGB 0-1) |
| `set_intensity(intensity)` | Change brightness |
| `set_radius(radius)` | Change light radius |
| `set_falloff(mode)` | Change attenuation curve |
| `set_spotlight(direction, angle)` | Enable spotlight mode |
| `set_direction(degrees)` | Rotate spotlight |
| `set_flicker(speed, amount, style)` | Configure flicker |
| `set_z(z)` | Height for normal mapping |
| `set_shadow_casting(enabled, softness)` | Enable/disable shadow casting |
| `get_position()` | Returns `(x, y)` |
| `get_color()` | Returns color tuple |
| `get_intensity()` | Returns base intensity |
| `get_radius()` | Returns radius |
| `compute_intensity(time)` | Intensity with flicker applied |

### LightingSystem Methods

| Method | Description |
|--------|-------------|
| `add(x, y, radius, color, ...)` | Add a light (returns `Light`) |
| `remove(light)` | Remove a light |
| `clear()` | Remove all lights |
| `render(time)` | Draw all lights (shadow + batched) |
| `set_renderer(renderer)` | Link renderer for shadow map generation |
| `get_light_count()` | Number of active lights |
| `is_full()` | True if at 256 lights |
| `release()` | Free GPU resources |

## FogLayer

Fullscreen procedural fog rendered via simplex noise. Render it inside the scene FBO with blending enabled so lighting illuminates the fog naturally.

```python
from pyluxel import FogLayer

fog = FogLayer(ctx)

# Configure persistent state
fog.set_color(0.6, 0.6, 0.65)
fog.set_density(0.4)
fog.set_wind_speed(0.5, 0.1)
fog.set_scale(3.0)
fog.set_height_falloff(1.5)   # 0=uniform, >0=denser at bottom

# Render (call during scene pass with blending enabled)
fog.render(
    time=app.time,
    color=(0.6, 0.6, 0.65),
    density=0.4,
    scale=3.0,
    height_falloff=1.5,
    wind_speed=(0.5, 0.1),
    design_size=(1280, 720),
)
```

### Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| `color` | `(0.6, 0.6, 0.65)` | Fog color (RGB 0-1) |
| `density` | `0.4` | Fog density (0-1) |
| `scale` | `3.0` | Noise pattern scale |
| `height_falloff` | `0.0` | Vertical falloff (0=uniform, 1-3=bottom-heavy) |
| `wind_speed` | `(0.5, 0.1)` | Wind velocity (vx, vy) |
| `sparsity` | `0.0` | Noise sparsity |

## ParticleSystem

GPU-batched 2D particle system with procedural SDF shapes. Supports up to **4096 particles** with burst and continuous emission, **8 shapes**, 3-stop colour gradients, spin, emission patterns, sub-emitters, and 10 built-in presets.

### Shape Constants

| Constant | ID | Description |
|----------|----|-------------|
| `SHAPE_CIRCLE` | 0 | Smooth circle with smoothstep edge |
| `SHAPE_SQUARE` | 1 | Rounded square |
| `SHAPE_SPARK` | 2 | Elongated along velocity with bright core |
| `SHAPE_RING` | 3 | Hollow circle (`ring_thickness` controls inner radius) |
| `SHAPE_STAR` | 4 | N-pointed star (`star_points`, `star_inner_ratio`) |
| `SHAPE_DIAMOND` | 5 | Diamond / rotated square |
| `SHAPE_TRIANGLE` | 6 | Equilateral triangle pointing up |
| `SHAPE_SOFT_DOT` | 7 | Gaussian falloff (softer than circle, great for fire/smoke) |

### ParticlePreset

Immutable configuration for particle emission. Define once, reuse everywhere.

```python
from pyluxel import ParticlePreset, SHAPE_SOFT_DOT

fire = ParticlePreset(
    # --- Emission ---
    count=20,                          # particles per burst (or per second if continuous)
    continuous=False,                  # True for continuous emission

    # --- Motion ---
    speed_min=50.0, speed_max=150.0,   # speed range (px/s)
    angle=270.0,                       # center direction (degrees, 0=right, 270=up)
    spread=30.0,                       # half-cone spread (degrees, 360=omni)
    gravity=0.0,                       # downward acceleration (px/s²)
    drag=0.5,                          # velocity damping (0=none, higher=faster stop)

    # --- Lifetime ---
    life_min=0.3, life_max=1.0,        # lifetime range (seconds)

    # --- Size ---
    size_start=8.0, size_end=2.0,      # size over lifetime (pixels)
    size_pulse_freq=0.0,               # Hz (0=disabled, sinusoidal oscillation)
    size_pulse_amount=0.0,             # amplitude in pixels

    # --- Colour ---
    color_start=(1.0, 0.6, 0.1, 1.0), # RGBA start color
    color_end=(1.0, 0.1, 0.0, 0.0),   # RGBA end color
    color_mid=(1.0, 0.3, 0.0, 0.8),   # RGBA mid color (None = 2-stop lerp)
    color_mid_point=0.4,               # normalised time of mid-color (0-1)
    intensity=1.5,                     # HDR colour multiplier (>1.0 for bloom glow)
    fade_in=0.1,                       # fraction of lifetime spent fading in (0-1)

    # --- Shape ---
    shape=SHAPE_SOFT_DOT,              # see shape constants above
    spark_stretch=3.0,                 # spark-only: length/width ratio
    ring_thickness=0.3,                # ring-only: inner radius ratio (0-1)
    star_points=5,                     # star-only: number of points (3-8)
    star_inner_ratio=0.4,              # star-only: inner/outer radius ratio

    # --- Velocity stretching (any shape) ---
    vel_stretch=0.0,                   # 0=no stretch, >0=elongate along velocity

    # --- Spin / rotation ---
    spin_min=0.0, spin_max=0.0,        # angular velocity range (degrees/sec, random direction)

    # --- Blending ---
    additive=True,                     # True=additive (ONE, ONE), False=alpha blending

    # --- Emission shape ---
    emit_shape="point",                # "point" | "ring" | "line" | "rect"
    emit_radius=0.0,                   # ring: radius
    emit_width=0.0,                    # line/rect: width
    emit_height=0.0,                   # rect: height
    emit_angle=0.0,                    # line: rotation angle (degrees)

    # --- Light emission ---
    emit_light=True,
    light_radius=60.0,
    light_intensity=0.5,
    light_color=None,                  # None = use color_start[:3]

    # --- Sub-emitter (on particle death) ---
    on_death=None,                     # ParticlePreset spawned when particle dies
    on_death_count=3,                  # particles per death event
)
```

### Built-in Presets

Ready-to-use presets that can be used directly or as starting points:

```python
from pyluxel import FIRE, SMOKE, EXPLOSION, SPARK_SHOWER, RAIN, SNOW, MAGIC, BLOOD, DUST, STEAM

particles.emit(400, 300, FIRE)
particles.emit(400, 300, EXPLOSION)
particles.emit_continuous("weather", 640, 0, RAIN)
```

| Preset | Shape | Additive | Key Features |
|--------|-------|----------|--------------|
| `FIRE` | soft_dot | yes | 3-stop colour, HDR 1.5x, fade-in |
| `SMOKE` | soft_dot | no | Growing size, fade-in |
| `EXPLOSION` | circle | yes | vel_stretch, HDR 2x, ring emit |
| `SPARK_SHOWER` | spark | yes | High gravity, HDR 1.8x |
| `RAIN` | spark | no | Continuous, line emit (1400px), fast |
| `SNOW` | soft_dot | no | Continuous, line emit, spin, size pulse |
| `MAGIC` | star | yes | 3-stop colour, spin, HDR 1.5x |
| `BLOOD` | circle | no | High gravity, fast drag |
| `DUST` | soft_dot | no | Slow, growing size, fade-in |
| `STEAM` | soft_dot | no | Growing size, slow rise, fade-in |

### Usage

```python
from pyluxel import ParticleSystem, FIRE, SMOKE

particles = ParticleSystem(ctx, max_particles=4096)

# Burst emission
particles.emit(x=400, y=300, preset=FIRE)
particles.emit(x=400, y=300, preset=FIRE, angle_override=180.0)

# Continuous emission (call every frame with current position)
particles.emit_continuous("torch_1", x=200, y=400, preset=SMOKE)
particles.stop_emitter("torch_1")

# Each frame
particles.update(dt)
proj = renderer._view_projection if renderer._view_projection is not None else renderer.projection
particles.render(projection_bytes=proj.tobytes())

# Light integration (feed to LightingSystem)
for lx, ly, lr, lc, li in particles.get_pending_lights():
    lighting.add(lx, ly, lr, lc, li)

# Management
particles.get_particle_count()
particles.is_full()
particles.clear_all_particles()
particles.clear_emitters()
particles.is_emitter_active("torch_1")
```

### Emission Shapes

Control where particles spawn relative to the emission point:

```python
# Ring: particles spawn along circle perimeter
ring_burst = ParticlePreset(emit_shape="ring", emit_radius=50, ...)

# Line: particles spawn along a line segment
rain = ParticlePreset(emit_shape="line", emit_width=1400, emit_angle=0, ...)

# Rect: particles spawn within a rectangle area
area = ParticlePreset(emit_shape="rect", emit_width=100, emit_height=50, ...)
```

### Sub-Emitters

Spawn child particles when a particle dies. Children cannot spawn further children (no recursion).

```python
smoke_trail = ParticlePreset(
    count=3, speed_min=5, speed_max=20, angle=270, spread=40,
    size_start=4, size_end=12, life_min=0.5, life_max=1.0,
    color_start=(0.5, 0.5, 0.5, 0.3), color_end=(0.3, 0.3, 0.3, 0.0),
    additive=False,
)

explosion = ParticlePreset(
    count=30, speed_min=100, speed_max=250,
    on_death=smoke_trail,    # each dying particle spawns smoke
    on_death_count=2,
    ...
)
```

### 3-Stop Colour Gradient

Add a mid-colour for richer colour transitions:

```python
flame = ParticlePreset(
    color_start=(1.0, 0.9, 0.3, 1.0),   # bright yellow
    color_mid=(1.0, 0.3, 0.0, 0.8),     # orange
    color_mid_point=0.4,                  # orange peaks at 40% lifetime
    color_end=(0.3, 0.05, 0.0, 0.0),    # dark red, fades out
)
```

### Velocity Stretching

Elongate any shape along its velocity direction (not just sparks):

```python
fast_circles = ParticlePreset(
    shape=SHAPE_CIRCLE,
    vel_stretch=2.5,   # 2.5x elongation along velocity
    ...
)
```

### Spin / Rotation

Particles can spin during their lifetime:

```python
sparkles = ParticlePreset(
    shape=SHAPE_STAR,
    spin_min=-180, spin_max=180,   # random rotation speed (degrees/sec)
    ...
)
```

Note: spin is applied to non-velocity-stretched particles only. Velocity-stretched particles orient along their movement direction instead.

## Transition

Scene transition effects rendered as a fullscreen overlay.

```python
from pyluxel import Transition, TransitionMode

transition = Transition(ctx)

# Start a fade-out (scene -> black)
transition.start(
    mode=TransitionMode.DISSOLVE,
    duration=0.5,
    color=(0.0, 0.0, 0.0),   # target color
    reverse=False,             # False=fade out, True=fade in
    on_complete=lambda: print("done!"),
)

# Each frame
if transition.active:
    transition.update(dt)
    transition.render(
        screen_texture=renderer.combine_texture,
        screen_width=renderer.screen_width,
        screen_height=renderer.screen_height,
    )

    if transition.done:
        # Switch scene, then fade back in
        transition.start(TransitionMode.DISSOLVE, 0.5, reverse=True)
```

### Transition Modes

| Mode | Description |
|------|-------------|
| `TransitionMode.FADE` | Simple alpha fade |
| `TransitionMode.DISSOLVE` | Noise-based dissolve |
| `TransitionMode.WIPE_LEFT` | Horizontal wipe from right to left |
| `TransitionMode.WIPE_DOWN` | Vertical wipe from top to bottom |
| `TransitionMode.DIAMOND` | Diamond-shaped reveal |

### Properties & Methods

| Member | Description |
|--------|-------------|
| `active` | True if a transition is running |
| `done` | True if transition reached the end |
| `progress` | Current progress (0.0 → 1.0) |
| `start(mode, duration, color, reverse, on_complete)` | Start transition |
| `stop()` | Stop immediately |
| `pause()` / `resume()` | Pause/resume |
| `get_mode()` | Current transition mode |
| `get_duration()` | Duration in seconds |
| `get_elapsed()` | Time elapsed |
| `is_reverse()` | True if fading in |

### Using with App

```python
app.start_transition(TransitionMode.FADE, duration=1.0)

# Check state
app.transition_active
app.transition_done
app.transition_progress

app.stop_transition()
```
